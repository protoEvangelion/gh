Motivation

Working with Github's UI can at times be painfully slow... especially when repetitive tasks could simply be executed from the CLI with much greater speed. That's why we created Node GH. To make tasks like opening, closing & forwarding Pull Requests a breeze!

Pull Requests

Issues

Create & Modify Github Issues without having to navigate back & forth in your browser

Repo

Open, create & clone repos with simple commands like "gh re"

It's easier than you think

One task that often takes up too much time is opening Pull Requests through the Github UI

gh pr --submit theUser --branch dev
1
 
1
gh pr --submit theUser --branch dev

Which can be simplified to:

gh pr -s theUser -b dev
1
 
1
gh pr -s theUser -b dev

It's as simple as that! But if you need fine-tuned control when submitting your PR (like linking it to an issue #) there are plenty other flags available.
